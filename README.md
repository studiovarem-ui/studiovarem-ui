# 🐉 드래곤 어드벤처 (Dragon Adventure)

아이들을 위한 인터랙티브 스토리 앱입니다. 아이가 직접 선택하며 이야기를 만들어가는 모험 게임!

![Dragon Adventure](https://img.shields.io/badge/React-19-blue) ![Tailwind CSS](https://img.shields.io/badge/Tailwind-3.4-38bdf8) ![License](https://img.shields.io/badge/License-MIT-green)

## ✨ 특징

- 🎮 **분기형 스토리**: 선택에 따라 달라지는 14가지 엔딩
- 🐉 **귀여운 용 친구**: 불꽃이, 반짝이, 구름이 중 선택
- 🏆 **엔딩 컬렉션**: 모든 엔딩을 수집하는 재미
- 🔊 **사운드 효과**: 버튼 클릭, 성공 시 효과음
- 📱 **모바일 최적화**: 어린이 손가락에 맞는 큰 버튼
- 💾 **진행 저장**: 로컬 스토리지에 자동 저장

## 🎯 14가지 엔딩

| 엔딩 | 설명 |
|------|------|
| 🎉 행복한 엔딩 | 불꽃이와 함께 집으로! |
| 🌟 우정 엔딩 | 요정과 친구가 되다 |
| 🏅 영웅 엔딩 | 용의 친구 메달 획득 |
| 🌸 꽃밭 엔딩 | 마법의 꽃을 선물받다 |
| 🍀 행운 엔딩 | 행운의 동전 발견 |
| 💎 보석 엔딩 | 우정의 보석을 받다 |
| 👑 왕족 엔딩 | 드래곤 왕국의 왕자/공주 |
| 🌅 노을 엔딩 | 아름다운 약속 |
| ⭐ 별빛 엔딩 | 별똥별 소원 |
| 🎉 파티 엔딩 | 구름 나라 파티 |
| 📸 추억 엔딩 | 영원한 기념사진 |
| 👨‍👩‍👧 가족 엔딩 | 용 가족의 일원 |
| 🎁 선물 엔딩 | 특별한 비늘 선물 |
| 🌠 소원 엔딩 | 별들의 축복 |

## 🚀 시작하기

### 설치

```bash
git clone https://github.com/studiovarem-ui/studiovarem-ui.git
cd studiovarem-ui
npm install
npm run dev
```

### 빌드

```bash
npm run build
```

## 🛠️ 기술 스택

- **React 19** - UI 라이브러리
- **Vite** - 빌드 도구
- **Tailwind CSS 3** - 스타일링
- **Web Audio API** - 사운드 효과
- **LocalStorage** - 진행 저장
